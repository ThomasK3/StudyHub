06 — Pravidla

Dos & Don'ts

Dělej ✓
  • Barlow Condensed 800 uppercase pro nadpisy
  • FIS Teal #00957d jako primární akci
  • Každá klauzule konzistentně svoji barvu
  • SQL vždy JetBrains Mono, tmavé pozadí
  • Border-radius max 6px (žádné pill buttons)
  • Step karty s border-left barvy klauzule
  • Deep teal (#003d35) pro footer a tabulkové hlavičky
  • Date badge přesně čtvercový jako FIS web
Nedělej ✗
  • Gradientová tlačítka (jen solid barva)
  • Inter / Roboto / system-ui písma
  • Červená mimo ORDER BY a chybové stavy
  • Pill tlačítka (border-radius 99px)
  • Teal jako barvu textu na bílém (kontrast)
  • Box-shadow blur větší než 12px
  • Vlastní barvy mimo definovanou paletu
  • Hero/banner bez dark overlay na fotce
